# hyper.nvim

A fork of [paulo-granthon's port](https://github.com/paulo-granthon/hyper.nvim) of the [Hyper.vim](https://github.com/jdsimcoe/hyper.vim) theme 

![Hyper.nvim example](./docs/example.png)

## Installation

### Using vim-plug

```lua
-- init.lua
Plug 'demoraescosta/hyper.nvim'
```
